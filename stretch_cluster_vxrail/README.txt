INTRODUCTION:
------------

This module contains a script which can be used to:
1.	Prepare a VxRail Cluster for Stretch operations
2.	Stretch a VxRail Cluster
3.	Expand a VxRail Stretched Cluster in SDDC.


REQUIREMENTS:
------------
For requirements, please refer to the 'VMware Cloud Foundation on Dell EMC VxRail Admin Guide'.


USAGE:
-----
For script usage, please refer to the 'VMware Cloud Foundation on Dell EMC VxRail Admin Guide'.
